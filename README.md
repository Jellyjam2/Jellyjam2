Black Swan is the strongest public engineering foundation in this project set.

##  Working Style

I build in small, verifiable slices:

```text
inspect → branch → patch → test → commit → PR → merge → release
```

Core principles:

- safety before speed
- clear logs before claims
- reproducible tests before public promises
- local-first workflows where possible
- no hidden system changes
- no secret keys in repos
